Installing Java and Python¶

Note

You can program your AI in Java or Python. However, Java is required to run the server regardless of what language you use to code your AI.

Note

JRE is enough if you are planning to make the AI in Python, but JDK is recommended.

Download the appropriate development kits from your profile page here. You will need to be logged into the dashboard.

1. Java (Required)¶

Note

You may get a Windows Security Alert when you first run the server. This is because it is communicating to the clients on a local port. Allow Java to communicate on the network when prompted. You will also get asked for administrative rights when running the scripts, which you have to say “yes” in order to run.

Ensure your version of Java is 1.8 or greater. To check your version of Java, run the following in the command line:

> java -version

If the version is below 1.8, download and install the latest version of JDK from Oracle. Choose the installation file according to your platform.

Note

Running The Challenge on a Windows machine and just installed Java? Follow the below steps.

  • Click the start button and search for “edit the system environment variables”.

  • Click the corresponding application (“Environment Variables”). Under System variables, select the “Path” variable and click edit.

  • Add the following.

    C:\\Program Files\\Java\\jdk1.8.0_101\\bin.

  • Next, look for “JAVA_HOME” and click “Edit…” If it is not listed, click “New…” and set the name to be “JAVA_HOME”.

  • Set the value to be the following.

    C:\\Program Files\\Java\\jdk1.8.0_101.

Warning

Note that the above JDK version may not be exactly the same as yours, but all of them would have the following syntax.

Java\\jdk1.X.X_XX\\bin.

Make sure you include the correct JDK version while editing your environment variables.

Note

Running The Challenge on a Windows machine and just updated Java? Follow the below steps.

  • Rerun the following.

    > java -version

  • If you get an error message about the registry key and/or java looking for a certain version and finding another, navigate to the following.

    C:\\Windows\\System32

  • There should be 3 files - java.exe, javaw.exe, javaws.exe - move these to another file location temporarily if you wish to revert to your previous version of Java.

  • Close and reopen the Command Prompt and confirm the correct version.

2. Python (Optional)¶

For this competition, we will be using Python 3.5 if you wish to code your AI in python.

Note

Python 3.5 or above is required, if you have an older version of Python. Upgrade here

To ensure you have Python 3.5 installed, run the following from the command line for windows and see if it says 3.5 and above:

> py -3

Or for Linux/Mac Systems:

> python3

Note

On Mac System, both commands may not work. Take a look here .

If Python is not a recognized command, download and install the appropriate version. Rerun this command to ensure that Python 3.5 or higher is now being used. If Python is installed but not working in command line, check the note below and edit your environment variable.

Note

Running the Orbis Challenge on a Windows machine and just installed Python, or Python is not included in your path? Follow the below steps to modify your Environment Variables.

  • Click the start button and search for “Edit the system environment variables” and click the corresponding application (“Environment Variables”).

  • Under System variables, select the “Path” variable and click edit.

  • If you are using Python, append the following.

    C:\\Program Files (x86)\\Python35-32;C:\\Program Files (x86)\Python35-32\\Scripts

  • Note that if you installed Python to a different folder, then you should append that one to your PATH rather than the above.

Warning

Note that the above Python version may not be exactly the same as yours, but all of them would have the following syntax.

Python35-XX

Make sure you include the correct Python version when editing your environment variables.
