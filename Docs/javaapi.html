Java API Basics¶

This page contains the essentials of the Java API. For the full API, see the Java Docs.

Essentials of the Essentials

Your task is to implement the:

doMove(World world, FriendlyUnit friendlyUnit, EnemyUnit[] enemyUnits)

function in the PlayerAI.java file. This function is invoked at each turn in the game. The PlayerAI class is instantiated just once at the beginning of the game, and that instance is used for the rest of the game.

Please be sure to give this a read prior to beginning the challenge!

Warning

Don’t change the function definition for doMove. This function is called by the client code (which you also shouldn’t modify) because when you submit your file, only your PlayerAI file and any additional libraries are kept and any client code is removed. Thus, if you change the method, your code will cause an Exception and ultimately invalidate all of your work.

1. Friendly and Enemy Units¶

// Both FriendlyUnit and EnemyUnit objects have methods that allow you discern its state.
// The EnemyUnit object has these same properties as well.
friendlyUnit.getPosition()
friendlyUnit.getStatus()
friendlyUnit.getBody()
friendlyUnit.getSnake()
friendlyUnit.getTerritory()
friendlyUnit.getTurnPenalty()

// Move your friendly_unit by calling the move method.
// For instance, the below moves your unit to the point (3, 3)
friendlyUnit.move(new Point(3, 3))

2. Points and Tiles¶

// The world instance contains a dictionary of points to tiles to help you find tile objects.
world.getPositionToTileMap.get(new Point(3, 3)) = // Tile at Point (3, 3)

// You can retrieve points from tiles as well.
tile.getPosition() = // Point (3, 3)

// Each tile contains useful information about the state of the tile, informing you of the
// ownership of the tile, as well as the presence of bodies / heads on the tile.
tile.getOwner() = // Team who owns the tile
tile.getBody() = // Team who owns the body on that tile (if any)
tile.getHead() = // Team who owns the head on that tile (if any)

// Each tile also contains useful booleans that should help you in navigating through the map.
tile.isNeutral() = // is the tile neutral?
tile.isEnemy() = // is the tile enemy territory?
tile.isFriendly() = // is the tile friendly territory?
tile.isWall() = // is the tile a wall?
tile.isEdge() = // is the tile at the edge of the map?

// These properties can be used to discern the state of the game - for example,
// you could determine whether or not you are on enemy territory by checking the following boolean.
currentTile = world.getPositionToTileMap.get(friendlyUnit.getPosition())
on_enemy_territory = currentTile.isEnemy()

3. World¶

// The world class contains methods that tell you information about the map.
world.getWidth()
world.getHeight()
world.getNeutralPoints()

// It also allows you to check if a point is in-bounds.
world.isWithinBounds(new Point(-1, -1)) = False
world.isWithinBounds(new Point(15, 15)) = True

4. World.TileUtils¶

// Within the World class, there is an instance of the TileUtils class.
// Using functions in this class will help you turn high level ideas into code fast!

// Get the closest capturable tile.
world.util.getClosestCapturableTerritoryFrom(friendlyUnit.getPosition(), null)

// Get the closest enemy body (perhaps for an attack!)
world.util.getClosestEnemyBodyFrom(friendlyUnit.getPosition(), null)

// Get the closest friendly territory, avoiding your own body.
// (perhaps to return to acquire territory!)
world.util.getClosestFriendlyTerritoryFrom(friendlyUnit.getPosition(), friendlyUnit.getSnake())

// Get the edges of your territory (perhaps to determine exposure, or plan!)
world.util.getFriendlyTerritoryEdges()

// Note that while friendly_unit.position is used as the argument for the examples above,
// you can use these functions to test any point on the map!

5. World.PathFinder¶

// Within the World class, there is an instance of the PathFinder class.
// Using functions in this class will help you find your way from one point to another!

// Find the shortest path from point a to point b, while avoiding points c.
world.path.getShortestPath(a, b, c)

// Find the next point in the shortest path from point a to point b.
world.path.getNextPointInShortestPath(a, b)

// Use these functions to devise strategies - for example, let's find a path to the
// nearest friendly territory, while avoiding points of our own snake.
currentPosition = friendlyUnit.getPosition()
closestFriendlyTile = world.path.getClosestFriendlyTerritoryFrom(currentPosition)
world.path.getShortestPath(currentPosition, closestFriendlyTile.getPosition(), friendlyUnit.getSnake())
