Welcome!¶

Welcome to the 2018 Orbis Challenge! For this year’s Orbis Challenge, you will create an AI for Snake, a four-player territory expansion game where you control a snake agent to capture territory against your opponent on a tile-based board. AIs must balance risk and reward for territory acquisition, while also devising strategies for encounters with opponents!

Note

Even if you are familiar with Java or Python, you should still read the documentation. The documents contain important information about the game, grading and competition rules.
