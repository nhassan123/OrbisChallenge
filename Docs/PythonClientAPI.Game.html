Represents a colour-changing tile on the board.

Variables:
  • is_neutral (boolean) – true if tile is neutral.
  • is_enemy (boolean) – true if tile is enemy territory.
  • is_friendly (boolean) – true if tile is friendly territory.
  • is_edge (boolean) – true if tile is at the edge.
  • is_wall (boolean) – true if tile is at wall.
  • owner (Team) – team that owns this tile.
  • body (Team) – team that has a body on this tile.
  • head (Team) – team that has a head on this tile.
  • position (tuple) – point corresponding to this tile.
